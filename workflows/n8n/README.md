# n8n Workflows

Export your n8n workflows as JSON and save them here.

## How to export

1. Open n8n
2. Open your workflow
3. Menu → Download
4. Save the `.json` file in this folder

## Workflows

| File | Description |
|------|-------------|
| `discord-ai-reply.json` | Discord message → Ollama AI → Discord reply |

## How to import

1. Open n8n
2. New workflow → Import from File
3. Select the JSON from this folder
